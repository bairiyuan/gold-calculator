# 黄金持仓均价计算器

一个简单易用的黄金持仓均价计算工具，支持积存金和黄金基金两种模块的计算。

## ✨ 功能特点

- **积存金模块**
  - 按克数（绝对值）计算
    - 按金额定投
    - 按重量定投
  - 按仓位百分比（相对权重）计算
- **黄金基金模块**
  - 按金额定投
  - 按份额定投
- **每日资讯**：AI生成的实时金融市场动态（集成DeepSeek API）
- **详细使用说明**：小白也能看懂的步骤指导
- **优雅界面**：温和的暖色调设计，支持自定义背景图片

## 🚀 运行方法

### 完整功能（推荐）

需要启动后端服务器来获取AI生成的资讯：

```bash
# 启动后端服务器
node backend-server.js
```

然后在浏览器中访问：`http://localhost:3000`

### 仅使用计算器（不获取资讯）

直接双击 `index.html` 文件在浏览器中打开即可。

## 📖 使用说明

### 积存金 - 按金额定投（最常用）

1. **第一步**：填【现有持仓重量（克）】→ 比如你现在有10克
2. **第二步**：填【现有均价（元/克）】→ 比如你现在10克的均价是700元
3. **第三步**：填【本次买入金额（元）】→ 比如你这次准备花7000元
4. **第四步**：填【本次买入价（元/克）】→ 比如现在金价是720元/克
5. **第五步**：点击【计算】按钮 ✨

**计算结果说明：**
- 新总重量：10克 + (7000÷720) ≈ 19.72克
- 新总成本：10×700 + 7000 = 14000元
- 新均价：14000 ÷ 19.72 ≈ 710元/克

### 积存金 - 按重量定投

当你知道要买多少克，而不是准备花多少钱时使用。

### 积存金 - 按仓位百分比

当你不知道具体有多少克，只想估算一下时使用（比如"假设我现在有100%的仓位，再买20%"）。

### 黄金基金

和积存金计算方式一样，只是把"克"换成"份"！
基金是按"份"买的，比如1份基金1.5元，你买1000元，就能买到约666.67份。

## 🎨 自定义配置

### 修改背景图片

将你的背景图片替换 `1ab203348206df1bd0d3810dedb31a73.png`，或修改 `style.css` 中的背景图片路径。

### 调整透明度

在 `style.css` 中修改：
- 页面整体透明度：`body::before` 的 `background` 属性（rgba的最后一个数值）
- 内容区域透明度：`.calculator-section` 和 `.news-section` 的 `background` 属性
